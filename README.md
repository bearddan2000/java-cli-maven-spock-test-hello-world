# java-cli-maven-spock-test-hello-world

## Description
A POC for maven app using spock framework.

## Tech stack
- java
- groovy
- maven
  - spock

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
